# Side Project

閒暇之餘也架了一個遊戲資料站 蘭德索爾圖書館 https://pcredivewiki.tw/ (opens new window)
其主要功能為查詢遊戲裝備，線性規劃未來刷裝建議以及其他各式計算小功能。

  • 從遊戲拆包到切版設計一人獨立完成。
  • 前台使用 Vue / Vuex / Vueroute / i18n / RWD 等技術的 Webpack Vue2.0 SPA 專案。
  • 後端使用一點PHP與MSSQL做為資料匯出用。
  • CDN 使用 Cloudflare 降低伺服器端的靜態資源負載。
  • 使用 bitbucket 管理 git

此專案從Domain、DNS、CND、前後端分離架構、DB 全部經歷過實為不錯的練習。
站台因中國引進此遊戲後，最高當月50萬人次，因遊戲慢慢不紅現平均為每月五萬人次。
